== Installation ==

1. Upload contents of the plugin zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In Admin Page: Flatsome > Theme Options > Style > Image Lightbox > uncheck "Enable Flatsome Lightbox" > Save
4. Add Element Gallery in UX Builder
